# Сравнение: BalisticNote Pro ↔ Applied Ballistics Quantum

Эталон — **AB Quantum** (мануал v3.7). Таблица отличий и план приведения «как в AB».
Статус: 🟢 как в AB · 🟡 частично · ⏸ отложено по решению пользователя · 🔴 нет.

| Область | Applied Ballistics Quantum | У нас (BalisticNote) | Статус |
|---|---|---|---|
| **Навигация** | Нижняя панель: Shoot · Profiles · Devices · Settings | Нижняя панель Стрельба/Профили/Полигоны/Настройки + боковое меню для остального | 🟢 |
| **Профили** | Отдельный экран, активный профиль карточкой, список, set active, edit, QR-шеринг | `/profiles` — список, активация тапом; `/profile/:id` — ОДИН экран (Bullet+Gun+Scope Data встроены), QR-шеринг («📤 Поделиться» → QR/ссылка → `/profile-import`) | 🟢 |
| **Профиль = что внутри** | Bullet Data + Gun Data + Scope + Advanced в одном объекте | `/profile/:id` редактирует всё это на одном экране; хранение по-прежнему через связку weapon+cartridge (не единый DB-объект) — сознательно, чтобы сохранить фичу «одно оружие + несколько патронов» | 🟡 |
| **Экран Shoot: структура** | Viewer (верх) + Controller (низ), свайп между контроллерами | HUD-карточка (Решение/Сетка) + закреплённый ветер W1/W2 + вкладки Meteo/Прочее, **свайп** между вкладками | 🟢 |
| **HUD: значения** | Elevation/Windage/Lead + % вероятности + раскрытие деталей | Вертикаль/Горизонталь (+W2) + строка TOF/энергия/скорость/Mach | 🟡 (без %, см. ниже) |
| **% вероятности / WEZ** | Есть в HUD | Убрано из HUD по прямому решению пользователя — «на самый последний вариант» | ⏸ |
| **Ветер** | «Откуда дует», компас/часы, stream с метеостанции | «Откуда дует», компас/часы (свайп на весь экран для точности), захват компасом телефона, **W1/W2 переключатель на главной**, **живой поток с Kestrel** (тумблер) | 🟢 |
| **Sight Scale Factor** | Есть (Ballistic Calibration → DSF) | Есть — `ssfElevation`/`ssfWindage` в оружии, применяется последним шагом ко всем поправкам | 🟢 |
| **Reticle (сетка)** | Онлайн-библиотека прицелов по брендам | Своя фото-сетка (загрузка+калибровка) + встроенная универсальная библиотека (МРАД-сетка/Mil-Dot/MOA-сетка, рисуются программно) — не бренд-специфично | 🟡 |
| **Bullet library** | Полная лабораторная база (калибр→бренд→вес) | ~35 пресетов (Berger/Sierra/Hornady/Lapua); .308 пополнен 5 записями из офиц. таблицы bergerbullets.com (проверено) | 🟡 |
| **Единицы измерения** | Настраиваемые (yd/m, MOA/MRAD, °F/°C…) | Зафиксированы константами по решению пользователя (м, mil, °C, гПа, граны, дюймы) — БЕЗ переключателей, см. `UNITS.md` | 🟢 (по решению пользователя, не переключаемо) |
| **Синхронизация** | AB Quantum Sync (облако) + QR + Bluetooth | Яндекс.Диск (merge без потерь) + QR-шеринг профилей | 🟢 |
| **Устройства (BT)** | Garmin, Kestrel 5700/x LiNK, Calypso AB Mini, хронографы | Kestrel 5700 (+ живой поток), SIG KILO (разовый замер) | 🟡 |
| **Солвер (физика)** | AJ, Coriolis, spin drift, SSF, CDM | То же + cant, powder-temp, DA, multi-point truing — паритет | 🟢 |
| **Graphs / Advanced Graphing** | Есть | Не делаем — пользователь сомневается в пользе без доплеровских данных | ⏸ |
| **Доп. инструменты** | Spotter AI, Learn (SOA), Range/Target Card editor | WEZ (код есть, скрыт из UI), анализ кучности (мм+mil), PRS-стейджи, Range Card печать, drag из DOPE, журнал+CSV, «Цели (брифинг)» — своя фича для соревнований | 🟡 |

## Открытая очередь (см. WORKLOG.md §8)
1. Дальномер SIG KILO — дистанция/азимут прямо в «Цели (брифинг)».
2. Экспорт/печать таблицы целей (как Range Card).
3. Переключатель ветра North↔DOF как явный тумблер (сейчас — через выбор Компас/Часы).
4. DSF-калибровка по доплеру — ждём, когда пользователь сможет прострелять.
5. WEZ / % вероятности / Graphs — отложено на «самый последний вариант».

## Известная находка (не исправлено, ждёт решения пользователя)
Два пресета в библиотеке пуль (`Berger 200.20X Hybrid`, `Berger 215 Hybrid`) имеют
длину пули (`len_in`), не совпадающую с официальной таблицей bergerbullets.com
(1.561/1.560 в коде vs 1.508/1.598 по офиц. данным). Похоже, эти записи внесены
раньше не из проверенного источника. Не трогали без запроса — см. WORKLOG.md §8b.5.
