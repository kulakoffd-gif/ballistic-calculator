# Сравнение: BalisticNote Pro ↔ Applied Ballistics Quantum

Эталон — **AB Quantum** (мануал v3.7). Таблица отличий и план приведения «как в AB».
Статус: 🟢 как в AB · 🟡 частично · 🔴 сильно отличается / нет.

| Область | Applied Ballistics Quantum | У нас (BalisticNote) | Статус | Что сделать |
|---|---|---|---|---|
| **Навигация** | Нижняя панель: Shoot · Profiles · (центр = выбор вида) · Devices · Settings | Боковое меню (≡ drawer) | 🔴 | Сделать нижнюю панель Shoot/Профили/Устройства/Настройки |
| **Экран Shoot: структура** | Viewer (верх) + Controller (низ). Вид Viewer переключается центр. кнопкой: HUD / Reticle / WEZ / Graphs | HUD-карточка + закреплённый ветер + вкладки Профиль/Meteo/Прочее | 🟡 | Виды HUD/Сетка/WEZ переключать в самом Viewer |
| **HUD: значения** | Elevation · Windage 1 · Lead · Windage 2 + **% вероятности** + раскрытие (TOF/энергия/скорость/Mach/Coriolis/Sg) | Вертикаль · Горизонталь (+W2) + строка TOF/энергия/скорость/Mach | 🟡 | Добавить % вероятности; Lead/Coriolis/Sg в раскрытие |
| **Контроллеры (низ)** | Выпадающий список: Environment / Target / Reticle / Ballistic Calibration / Speedtracker… свайп влево-вправо | Вкладки Профиль / Meteo / Прочее + закреплённый ветер | 🟡 | Привести к набору контроллеров AB |
| **Профили** | Отдельный экран: активный профиль (карточка Bullet/BC/MV/Zero/SightHeight/Twist), список, **группы**, поиск, сортировка, list/card, set active, edit, duplicate, delete, **QR-шеринг**, облачная синхр., отправка на устройства | Раздельно «Оружие» и «Патроны»; выбор активного — на /calc; нет единого профиля, групп, QR | 🔴 | Отдельный экран «Профили», профиль = оружие+патрон+прицел, выбор активного там, не на Shoot |
| **Профиль = что внутри** | Один объект: Bullet Data + Gun Data + Scope + Advanced + MV-Temp/DSF | Раздельно weapon (твист/высота/нолевая/сетка) + cartridge (BC/масса/V₀/…) | 🟡 | Возможно объединить в один «профиль» |
| **Выбор оружия/высоты прицела** | Только в профиле (один раз), на Shoot не редактируется — вверху лишь имя активного профиля | Вынесено в сводку «Профиль» на /calc; ручной ввод скрыт без профиля | 🟡 | Имя активного профиля в шапке Shoot, выбор — на экране «Профили» |
| **Ветер** | «Откуда дует», компас/часы, переключатель North↔DOF (Compass/Clock reference), stream wind с метеостанции | «Откуда дует», компас/часы, захват компасом телефона | 🟡 | Добавить переключатель North/DOF; stream wind (Kestrel) |
| **Reticle (сетка)** | Онлайн-библиотека прицелов (бренды/модели), авто-рисование сетки + точка решения, subtension | Загрузка фото сетки + калибровка 2 точками | 🔴 | Библиотека сеток — большой объём; пока остаётся фото |
| **Bullet library** | Лабораторная база пуль (калибр→бренд→вес), CDM/PDM | ~30 пресетов + custom drag вручную | 🟡 | Расширять пресеты; CDM вводится вручную |
| **Калибровка** | MV Cal / CDF / DSF (матрица выбора) | BC truing (1 и многоточечный) + MV-Temp таблица | 🟡 | Добавить DSF (drop scale factor) при желании |
| **Единицы измерения** | Настраиваемые: yards/meters, FPS/MS, MOA/MRAD, °F/°C, inHg/mbar, grains/grams | В основном метрические фикс. (м, м/с, mil+MOA, °C, гПа, gr) | 🔴 | Экран настроек единиц |
| **Синхронизация** | Облачный аккаунт AB Quantum Sync + QR + Bluetooth | Яндекс.Диск (свой merge без потерь) | 🟡 | Своя реализация, работает; QR-шеринг профилей — можно добавить |
| **Устройства (BT)** | Garmin, Kestrel 5700/x LiNK, Calypso AB Mini, хронографы, E-Dope | Kestrel 5700, SIG KILO (BLE), правка парсера | 🟡 | Зависит от железа |
| **Солвер (физика)** | AJ, Coriolis, spin drift, SSF, CDM (Elite/Pro) | G1/G7/custom, RK4, Coriolis, spin drift, AJ, cant, powder-temp, DA, multi-point truing | 🟢 | Паритет по эффектам |
| **Доп. инструменты** | Spotter AI, Learn (SOA), Graphs/Advanced Graphing, Range/Target Card column editor, Grouped Sectors, Sight Scale Factor | WEZ Monte-Carlo, анализ кучности по фото, PRS-стейджи, Range Card печать, drag из DOPE, заметки, подготовка гильз, журнал+поиск+CSV | 🟡 | Разные сильные стороны; графиков нет |

## Главные расхождения (приоритет для приведения к AB)
1. 🔴 **Навигация** — нижняя панель вместо бокового меню.
2. 🔴 **Отдельный экран «Профили»** — выбор активного профиля там, а не на Shoot;
   на Shoot вверху только имя активного профиля (тап → Профили).
3. 🟡 **Виды Viewer** (HUD / Сетка / WEZ) переключать внутри верхней карточки.
4. 🟡 **% вероятности попадания** в HUD.
5. 🔴 **Настройка единиц** (yd/m, MOA/MRAD, °F/°C и т.д.).
