# Сравнение: BalisticNote Pro ↔ Applied Ballistics Quantum

Эталон — **AB Quantum** (официальный User Manual, Rev 2, 03/2026, 42 стр. — прочитан
полностью). Живой документ: сверяем и актуализируем по ходу разработки.

Визуальная версия (таблица с фильтрами, счётчиками): [артефакт](https://claude.ai/code/artifact/5a87a89e-af69-4361-9cd8-3f6328c8c785)
— если ссылка протухнет, попроси Claude пересобрать её из этого файла.

**Статусы:**
- 🟢 **паритет/лучше** — делаем так же или лучше, чем AB
- 🟡 **частично** — функция есть, но не эквивалентна
- ⏸ **отложено** — технически реализуемо, отложено нашим решением
- ⚪ **можно, не приоритет** — реализуемо, но низкая ценность/эффорт не оправдан
- 🔴 **не выйдет** — блокируется внешним ограничением (чужие данные/протокол/контент)

## 01. Навигация и профили

| Функция | AB Quantum | BalisticNote Pro | Статус | Комментарий |
|---|---|---|---|---|
| Нижняя навигация | Shoot · Profiles · Devices · Settings | Стрельба · Профили · Полигоны · Настройки (+ боковое меню) | 🟢 | — |
| Единый профиль на одном экране | Bullet+Gun+Scope Data на одной странице | То же — `/profile/:id` | 🟢 | — |
| Список профилей: сортировка, list/card, папки | Есть | Список + активация тапом; без папок/сортировки | 🟡 | Малый эффект при личном использовании |

## Быстрые возможности (низкий эффорт, стоит рассмотреть)
- Интерактивный калькулятор SSF (Tall Target Test) — будет лучше, чем у AB.
- Графики Path/Custom — не требуют доплера, можно строить на текущем солвере.
- Fire Control тумблеры (AJ/Spin Drift/Coriolis on/off).
- Явный тумблер North/DOF для ветра.
- Column Editor для Range/Target Card.

## Полностью заблокировано (не пытаться без смены обстоятельств)
- Брендовая онлайн-библиотека сеток, лабораторная база пуль по доплеру.
- Live-протоколы Garmin/Calypso/SpeedTracker/Xero/E-Dope — закрытые/неподтверждённые протоколы.
- Learn (SOA) — требует создания/лицензирования видео-контента.
