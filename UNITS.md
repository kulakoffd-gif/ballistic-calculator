# Единицы измерения — где что используется

Заполни колонку «Выбор». ✅ — ты уже подтвердил. ❓ — нужно твоё решение.
Меняется один раз (по сути на входе), пересчёт на лету не нужен.

| Величина | Где встречается | Сейчас | Твой выбор |
|---|---|---|---|
| Дистанция / дальность | расчёт, таблица, цели, нолевая | м | ✅ метры |
| Температура (воздух, боеприпас) | Meteo | °C | ✅ °C |
| Давление | Meteo | гПа (=мбар) | ✅ гПа/мбар |
| Поправки (вертикаль/горизонталь/ветер) | HUD, таблица, сетка | mil (+ MOA + см рядом) | ✅ mil — **убрать MOA и см?** ❓ |
| Вес пули | патрон | гран (gr) | ✅ граны |
| Навеска пороха | патрон | гран | ✅ граны |
| Длина пули | патрон | дюйм (in) | ✅ дюймы |
| Калибр пули | патрон | дюйм (in) | ❓ дюйм / мм? |
| Скорость пули V₀ | патрон, Быстро | м/с | ❓ м/с / fps? |
| Скорость ветра | Meteo, ветер | м/с | ❓ м/с / fps / mph? |
| Высота прицела над каналом | оружие | мм | ❓ мм / см / дюйм? |
| Твист ствола | оружие | дюйм (1 оборот / N in) | ❓ дюйм / мм? |
| Энергия пули | детали HUD | Дж (+ ft·lb) | ❓ Дж / ft·lb / оба? |
| Density Altitude | детали HUD, атмо | фут (ft) | ❓ фут / м? |
| Размер цели (кучность, WEZ) | анализ кучности | см | ❓ см / мм / МОА? |
| Скорость движущейся цели | движущаяся цель | м/с (или mil/с) | ❓ м/с по умолч.? |
| Углы (азимут, угол места, завал) | расчёт | градусы (°) | ✅ градусы |
| Время полёта (TOF) | детали HUD | сек | ✅ сек |
| Цена клика барабана | оружие/сетка | mil (0.1) | ❓ mil / MOA? |

Допиши/исправь — встрою как значения по умолчанию (один раз, без переключателей).
