# Единицы измерения — зафиксировано в коде (`www/units.js`)

Это справочный файл, НЕ экран в приложении. Единицы прописаны в коде один раз
как константы (без переключателей и конвертера — по решению пользователя;
конвертер единиц можно добавить в будущем, если понадобится).

| Величина | Где встречается | Единица |
|---|---|---|
| Дистанция / дальность | расчёт, таблица, цели, нолевая | ✅ метры |
| Температура (воздух, боеприпас) | Meteo | ✅ °C |
| Давление | Meteo | ✅ гПа/мбар |
| Поправки (вертикаль/горизонталь/ветер) | HUD, таблица, сетка | ✅ mil (MOA и см убраны) |
| Вес пули / навеска пороха | патрон | ✅ граны |
| Длина пули / калибр | патрон | ✅ дюймы |
| Скорость пули V₀ | патрон, Быстро | ✅ м/с |
| Скорость ветра | Meteo, ветер W1/W2 | ✅ м/с |
| Высота прицела над каналом | оружие | ✅ мм |
| Твист ствола | оружие | ✅ дюймы |
| Энергия пули | детали HUD | ✅ Дж |
| Density Altitude | детали HUD, атмо | ✅ метры |
| Размер цели / кучность (Mean Radius, ES) | анализ кучности | ✅ мм + mil (см и MOA убраны) |
| Скорость движущейся цели | движущаяся цель | ✅ м/с и mil/с — переключатель прямо на экране (уже был) |
| Углы (азимут, угол места, завал) | расчёт | ✅ градусы |
| Время полёта (TOF) | детали HUD | ✅ сек |
| Цена клика барабана | оружие/сетка | ✅ mil |

Если что-то нужно поменять — правишь эту таблицу и говоришь мне, я переношу
значение в `www/units.js` руками. В приложении экрана настройки единиц нет.
